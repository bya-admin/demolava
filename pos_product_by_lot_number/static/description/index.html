Adding Lot/Serial Number to the products can reduce the manual efforts of the cashier in the Point of Sale. These numbers help the cashier to easily add the products in the cart.
Now, you can add the Lot/Serial Number in the products of Odoo POS. The module Odoo POS Scan Product By Lot/Serial Number allows the Odoo user to scan and add the products to the Odoo POS cart/order using Lot/Serial numbers of the products. The POS user can scan the lot/serial number using a barcode scanner.
Track the Products with the help of Lot/Serial Number

Now, you can easily add the products along with its Lot/Serial Number. Hence, it reduces the dependency on Barcode for making a product entry in the POS.

The module simply facilitates the POS user to use the Lot/Serial Number to Scan Product in Odoo POS and add the products in the cart in the running POS session.

When the POS user scans the products, a pop-up screen shows the lot number(s) associated with the products. The user can select the lot/serial number or manually enter a new one to add the product to the POS cart.

Detailed Features List

Below is the detailed list of Feature for Odoo POS Scan Product By Lot/Serial Number

  • The module facilitates the Odoo POS user to add the product to the Odoo POS cart using a lot/serial number of the product.
  • When a product is scanned or selected in Odoo POS, a pop-up shows the list of lot numbers or serial numbers associated with the products.
  • The Quantities in the Lot are automatically updated whenever a product is validated, added or removed from an order.
  • A product quantity higher than that in the chosen lot cannot be added to the cart.
  • If the quantity entered into the POS cart/order is greater than the number of products in the lot then a pop up will be raised which indicates the insufficient quantity.
  • If a wrong lot/serial number is entered manually, the system displays an error message.
